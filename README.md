## React Movie Library v2

## Tech Tools for building the app

- React
- Typescript
- React Query
- Chakra UI
- Vite
- Zustand for global state management

### Start Project

```
git clone https://github.com/jrussbautista/react-movie-library-v2
cd react-movie-library-v2
npm run dev
```

### Run Tests

```
npm run test
```
